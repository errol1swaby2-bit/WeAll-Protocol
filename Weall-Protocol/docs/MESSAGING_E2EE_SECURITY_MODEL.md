# Messaging E2EE Security Model

Status: unsupported.

The public-only WeAll protocol no longer includes a protocol-native encrypted messaging security model. WeAll must not store encrypted direct-message envelopes, private threads, or opaque encrypted social payloads with protocol meaning.

All protocol-native social, civic, governance, moderation, dispute, group, reputation, and validator/operator activity must remain publicly inspectable. Any private draft exists only in the local client before publication. External private communication tools are outside protocol scope.
