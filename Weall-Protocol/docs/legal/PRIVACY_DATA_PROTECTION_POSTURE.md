# Privacy and data-protection posture

Status: non-lawyer draft, pending counsel review.

The protocol includes identity, Proof-of-Humanity, review, dispute, public activity, and media surfaces. Public beta requires a counsel-reviewed privacy posture covering evidence retention, user deletion/export expectations, minors, sensitive media, reviewer access, logs, IPFS/media storage, and cross-jurisdiction processing.

User-to-user communication tooling is outside protocol scope; protocol group content remains publicly readable while membership gates participation.
