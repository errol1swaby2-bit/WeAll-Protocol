# Code of Conduct

WeAll is intended to support serious technical collaboration around decentralized coordination, identity, governance, and dispute systems.

This project welcomes participation from people with different backgrounds, skill levels, and perspectives. Productive disagreement is allowed. Harassment, intimidation, and personal attacks are not.

## Expected behavior

Participants are expected to:

- be respectful and specific
- critique ideas and code rather than people
- assume good faith when reasonable
- give clear reproduction steps for bugs
- keep security-sensitive details private
- respect boundaries when maintainers pause or redirect a discussion

## Unacceptable behavior

Unacceptable behavior includes:

- harassment, threats, or intimidation
- discriminatory or degrading remarks
- doxxing or sharing private information without consent
- sexualized or violent comments directed at participants
- spamming, trolling, or repeated bad-faith disruption
- publishing exploit details after being asked to use the private security process

## Enforcement

Maintainers may moderate, edit, hide, lock, or remove comments, issues, pull requests, or other participation that violates this code.

For severe or repeated violations, maintainers may restrict participation in the project.

## Scope

This code applies to project spaces, including GitHub issues, pull requests, discussions, documentation, and related technical review channels.
